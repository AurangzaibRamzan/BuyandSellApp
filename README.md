## :rocket: Installation
1. Get the code

    ```bash
    git clone https://github.com/AurangzaibRamzan/BuyandSellApp.git && cd BuyandSellApp
    ```

2. Install dependencies

    ```bash
    npm install
    ```
3. In terminal, run

    ```bash
    npm start
    ```
3. In separate terminal, run the project

    ```bash
    # for iOS
    react-native run-ios
    ```

    ```bash
    # for android
    react-native run-android
    ```
